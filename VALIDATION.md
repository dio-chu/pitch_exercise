# 頻率檢測驗證指南

## ✅ 已完成的更新

### 1. 半音點擊發聲

- 所有音符（包括半音）現在都可以點擊播放
- 在 Western (ABCDEFG) 模式下，所有音符都顯示標籤並可點擊
- 在 Solfege (Do Re Mi) 和 Number (1234567) 模式下，只有調內音可點擊

### 2. 頻率檢測驗證

已經驗證以下公式的正確性：

#### MIDI ↔ 頻率轉換

```javascript
// MIDI → 頻率
frequency = (440 * 2) ^ ((midi - 69) / 12);

// 頻率 → MIDI
midi = 12 * log2(frequency / 440) + 69;
```

#### 標準音高對照表

| 音符 | MIDI | 頻率 (Hz) |
| ---- | ---- | --------- |
| C2   | 36   | 65.406    |
| C3   | 48   | 130.813   |
| C4   | 60   | 261.626   |
| A4   | 69   | 440.000   |
| C5   | 72   | 523.251   |
| C6   | 84   | 1046.502  |
| C7   | 96   | 2093.005  |

#### 音高顯示計算

```javascript
// Y 座標 = (最高音 MIDI - 當前音 MIDI) * 每行高度
y = (96 - midiFloat) * 22;

// 例如：
// C7 (MIDI 96) → y = 0px (最上方)
// C4 (MIDI 60) → y = 792px (向下 36 個半音)
```

## 🧪 驗證方法

### 方法 1：點擊音符測試

1. 切換到 "ABCDEFG" 模式
2. 點擊任意音符標籤
3. 查看瀏覽器控制台（F12 → Console）
4. 應該看到類似：`🎵 Playing: MIDI 60, Frequency 261.63 Hz`
5. 播放的音高應該與紅色橫線的位置完全對齊

### 方法 2：麥克風測試

1. 點擊 "Detect" 開始偵測
2. 唱一個穩定的音（例如 C4 ≈ 262 Hz）
3. 觀察：
   - 藍色軌跡線應該穩定在對應的音高
   - 紅色橫線標註當前音高
   - 頂部顯示頻率和音分偏差

### 方法 3：對照測試

1. 開啟 `test-frequency.html` 測試頁面
2. 點擊播放測試音符
3. 同時開啟 Pitch Trainer 並唱相同的音
4. 確認兩者顯示的頻率一致

### 方法 4：音叉/調音器對照

如果你有物理音叉或電子調音器：

1. A4 音叉 = 440 Hz
2. 在 Pitch Trainer 中應該顯示為 MIDI 69
3. 偏差應該在 ±10¢ 以內

## 🎯 預期結果

### 正確的行為

- ✅ 點擊 C4 標籤時播放 261.63 Hz
- ✅ 麥克風檢測到 440 Hz 時顯示 A4
- ✅ 藍色軌跡線和紅色橫線對齊在音符標籤的位置
- ✅ 點擊播放音符後，紅線應該穿過該音符的橫線

### 可能的誤差來源

1. **麥克風品質**：內建麥克風可能有頻率響應不均的問題
2. **背景噪音**：環境噪音會影響偵測準確度
3. **人聲特性**：人聲包含泛音，可能導致八度跳躍
4. **音準偏差**：自然唱歌通常會有 ±10-20¢ 的偏差

## 📊 技術細節

### YIN 演算法

- 使用 YIN 音高檢測演算法（de Cheveigné & Kawahara, 2002）
- FFT 大小：4096 samples（適合低音檢測）
- 檢測範圍：C2 (65 Hz) ~ C7 (2093 Hz)
- 閾值：0.18（較寬鬆，適合人聲）

### 八度穩定性

- 內建八度跳躍修正
- 如果新頻率與前一個頻率相差 > 8 半音，檢查 ×2 和 ÷2 是否更接近
- 有效減少八度誤判

### Cents 計算

- 1 個半音 = 100 cents
- cents = (midiFloat - midiRounded) × 100
- 範圍：-50¢ ~ +50¢

## 🐛 已知限制

1. **極低音/極高音**：超出 C2~C7 範圍的音可能無法準確檢測
2. **和弦/複音**：只能檢測單音，和弦會導致不穩定
3. **顫音**：快速顫音會顯示為抖動的軌跡線
4. **環境音**：背景音樂或噪音會干擾檢測

## 💡 建議

- 在安靜的環境中測試
- 使用耳機或外接麥克風以獲得更好的效果
- 唱穩定的長音，避免快速移動音高
- 參考頂部顯示的 cents 值來微調音準
